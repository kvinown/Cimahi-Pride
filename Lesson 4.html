Form & PHP Dasar

Materi yang akan kita pelajari:
  • Form
    		•
  • form>
    		•
    Tag form memiliki fungsi untuk mengumpulkan input yang di isi oleh user yang di kirim kepada server.
    Beberapa attribute yang ada dalam tag formm:
    1. name
      2. Fungsi dari name untuk menamai form tersebut.
    2. method
      3. Fungsi dari method untuk menentukan form tersebut ingin menggunakan metode spesifik yang mana.
    3. action
      4. Fungsi dari action untuk menentukan form tersebut ingin bereaksi pada file mana. (*Untuk materi sekarang akan bereaksi dengan bahasa pemrogramman PHP)
    4. target
