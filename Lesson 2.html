  • <font>

    		•
    Tag font memiliki fungsi untuk mengubah tampilan font yang akan ditampilkan.
    Beberapa attribute yang ada dalam tag font:
    1. face
      2. Memiliki fungsi untuk emngubah jenis font yang akan digunakan
    2. color
      3. Memiliki fungsi untuk mengubah warna font yang akan ditampilkan
    3. size
      4. Memiliki fungsi untuk mengubah ukuran font yang akan ditampilkan
  • <a>

    		•
    Tag a memiliki fungsi sebagai hyperlink atau untuk link dari satu halaman ke halaman lainnya.
    Beberapa attribute yang ada dalam tag a:
    1. href
      2. Memiliki fungsi untuk menempatkan destinasi halaman yang akan dituju. (*attribute terpenting dalam tag a, karena jika tidak terdapat attribute href tag a tidak akan berfungsi.)
    2. type
      3. Memiliki fungsi untuk menentukan tipe media yang akan dihubungkan.
    3. target
      4. Memiliki fungsi untuk menentukan dimana tempat spesifik saat kita menekan link.
    4. download
      5. Mmemiliki fungsi untuk mengunduk/mengdownload target saat user mmengklik link.
  • <img>

    		•
    Tag img memiliki fungsi sebagai menampilkan gambar pada halaman
    Beberapa attribute yang ada dalam tag img:
    1. src
    2. alt
    3. width
    4. height
    5. title
    6. border
    7. hspace
    8. vspace
  • <marquee>

    		•
    Tag marquee membuat tampilan bergerak vertikal atau horizontal.
